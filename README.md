# 🍳 DataSynth-Cookbook

> 注重代码实现、学习与实用性的数据合成方法合集

**DataSynth-Cookbook** 收集并复现业界优秀的**数据合成（Data Synthesis）方法**。
不同于单纯的论文清单，这里的每个方法都有**完整的代码实现**，
让你既能**学懂原理**，又能**直接拿来用**。

## ✨ 三大核心

- 💻 **代码实现**：每个方法都有可运行的完整代码，不止于理论
- 📖 **学习友好**：配套原理讲解与注释，看得懂、学得会
- 🚀 **实用导向**：聚焦真实业务场景，开箱即用、快速落地
